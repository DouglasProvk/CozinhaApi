# ? Quick Start

## ?? Iniciar o Projeto

### Backend

```bash
# Na raiz do projeto
dotnet run

# Ou selecione "https" no Visual Studio e aperte F5
```

O backend estará em: `https://localhost:7000`
Swagger: `https://localhost:7000/swagger`

### Frontend

```bash
# Em outro terminal
cd Cozinha-FrontEnd
npm run dev
```

O frontend estará em: `http://localhost:5173`

---

**Pronto! Backend + Frontend rodando juntos com CORS configurado.** ?
