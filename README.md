# 🍽️ CozinhaAPI

API RESTful desenvolvida em .NET 8 + React para gerenciar pratos e reservas de uma cozinha.

## ⚡ Quick Start

### Backend
```bash
dotnet run
```
Backend em: `https://localhost:7000`

### Frontend
```bash
cd Cozinha-FrontEnd
npm run dev
```
Frontend em: `http://localhost:5173`

## Características

* Backend .NET 8 com Entity Framework Core
* Frontend React com Vite
* CRUD completo para Pratos e Reservas
* Swagger/OpenAPI
* CORS configurado
* SQL Server
